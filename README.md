# Nuxt 3 with FormKit + Storybook

Basic example of using FormKit with Nuxt 3 with Storybook

## Setup

```bash
# install dependencies
yarn install

# run dev nuxt
yarn dev

# run storybook
yarn storybook
```

## Tech Stack

- [Nuxt 3](https://v3.nuxtjs.org/)
- [FormKit](https://formkit.com/)
- [Storybook](https://storybook.js.org/)
